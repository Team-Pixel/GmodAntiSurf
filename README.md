# GmodAntiSurf
A simple anti prop-surf on Garry's Mod.

## Convars

__antipropsurf_enable__: 1 to enable the anti prop-surf, 0 to disable
__antipropsurf_admin_prevent__: 1 to enable the anti prop-surf for admins, 0 to disable
